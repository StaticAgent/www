Hi. I’m F. Emerson, a.k.a. StaticAgent.

I’m a software engineer and aspiring game programmer, currently working at Aquiris Game Studio, in Porto Alegre. I have been developing software professionally since 2015, focusing mostly on systems programming, server-side web development, database design and game server programming.

I’m an adept of building software the handmade way. I believe in developing software that works with the computer, not against it. In building things with the care of a craftsman and the attention to details of an engineer. In striving to find the most efficient and usable solution with the restrictions at play. One can only do so by putting effort into understand what it going on in the code and what happens when it interacts with the hardware.

Working this way makes one enter a virtuous circle: caring to undestand how the software works and how it interacts with the environment around it enhances one’s knowledge about the software and its environment, which in turn results in an even better software, software that is featureful, reliable and efficient.

Regarding my academic background, I hold a B.Sc. degree in Computer Science from the Federal University of Paraíba (UFPB).

My main research interests include: high-performance realistic rendering, combinatorial optimization, metaheuristics, parallel computing, and high-performance computing. I’m also interested in artificial intelligence, digital image processing, computer vision, robotics, and formal languages.
